# Phase2 Review - Wuhu Island Reconstruction

## 現在のv0.3スクリーンショットから見えた問題

1. 島の輪郭が円形に近く、公式マップのような複雑な入り江・岬・砂州が弱い。
2. 南側の白砂ビーチは良いが、単なる横長の帯になっており、南東へ伸びる特徴的な地形がない。
3. 湖は見やすいが、上流・滝・川の導線がないため、地形としての意味が薄い。
4. 町が小さく、道・広場・港との関係が弱い。
5. 風車、灯台、火山、森は配置されているが、エリア同士の移動導線がまだ薄い。
6. ミニマップは地形の大枠表示として機能しているが、道・離島・湖の複数構造が足りない。

## v0.4で入れた改善

- `islandOutline` を23点の非対称輪郭に変更。
- `secondaryIslands` を追加し、離島・岩礁をデータ化。
- `upperLake` と `river` を追加し、中央湖への水の流れを表現。
- `pathRoutes` を追加し、町・湖・風車・遺跡・海岸を道で接続。
- `heightAt()` を拡張し、火山・高地・湖・砂浜・町・離島を別々に整形。
- `addPathRoutes()` で道を3D上に表示。
- `drawMinimap()` で道・離島・上流湖を表示。

## 次のPhase2.5でやるべきこと

1. 建物パーツを増やす
   - ホテル
   - 広場の噴水
   - 桟橋の柱
   - カフェ風の小屋

2. 自然物を増やす
   - 岩の種類
   - ヤシ風の木
   - 滝の水柱
   - 浅瀬の水色グラデーション

3. 探索導線を強くする
   - 道に沿ってランドマークを置く
   - リングコースを道・湖・火山に沿わせる
   - 遠くから見える巨大目印を増やす

4. パフォーマンス設計
   - 遠くの木を非表示
   - 遠方の建物を簡略化
   - ランドマーク球は距離でサイズ/透明度を変える

## 重要方針

公式素材やモデルを直接使わず、位置関係・探索体験・地形構造を参考にしてオリジナル化する。
