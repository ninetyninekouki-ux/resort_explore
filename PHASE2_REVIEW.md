# Phase2 Review / v0.5

## 現状の問題

v0.4は島として動作していたが、輪郭と起伏がまだ公式マップ的な印象から遠かった。主因は `islandOutline` が存在していても、実際の高さ生成で楕円ベースの `edgeFalloff()` を使っていたため、地形が丸く見えていたこと。

## v0.5での修正

### 1. 輪郭生成方式

修正前：

```js
const radial = Math.sqrt((x / 310) ** 2 + (z / 300) ** 2);
```

修正後：

```js
const shore = distanceToPolygonEdge(x, z, islandOutline);
return smoothstep(0, 42, shore);
```

これにより、地形の海岸線が `islandOutline` に忠実に従う。

### 2. 輪郭再設計

- 西側に岬と岩場
- 北側に山体・遺跡・山荘
- 中央に湖
- 南に町と港
- 南東に長い砂州
- 南東沖に離島

### 3. 起伏再設計

- 火山は `volcano` と `volcanoShoulder` の二重構造
- 湖は強制的に低く削る
- 上流池から中央湖へ川・滝を設置
- 町・広場・港は平坦化
- 砂浜は低地へ平坦化
